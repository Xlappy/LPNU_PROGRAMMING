# Лабораторна робота №2: Робота з RFID MFRC522 (Бібліотека MFRC522)

Ця лабораторна робота фокусується на взаємодії з RFID-модулем MFRC522 по протоколу SPI для зчитування ідентифікатора (UID) та запису даних у пам'ять мітки (MIFARE Classic 1K).

## Схема підключення (Arduino Uno)
| MFRC522 | Arduino Pin | Провід |
|:-------:|:-----------:|:------:|
| VCC     | 3.3V        | Червоний (ВАЖЛИВО: не 5V!) |
| RST     | D9          | Будь-який |
| GND     | GND         | Чорний |
| MISO    | D12         | SPI |
| MOSI    | D11         | SPI |
| SCK     | D13         | SPI |
| SDA(SS) | D10         | SPI |
| Green LED| D7          | Output |
| White LED| D8          | Output |

[Link to Datasheet (PDF)](datasheets/MFRC522_Datasheet.pdf)

## Логіка роботи з LED
1. **Режим очікування**: Горить **Білий світлодіод** (Піни 8).
2. **Програмування мітки**: Як тільки мітку знайдено, **Білий** починає моргати (6 разів), імітуючи процес запису.
3. **Успішне зчитування/запис**:
   - Білий гасне.
   - **Зелений світлодіод** (Пін 7) засвічується на **5 секунд**.
   - Система повертається в режим очікування (Зелений гасне, Білий вмикається).

## Схема підключення LED
Для LED використовуйте ваші резистори **110 Ом**:
- **Arduino D7** -> Резистор -> Довга ніжка (+) Зеленого LED.
- **Arduino D8** -> Резистор -> Довга ніжка (+) Білого LED.
- Короткі ніжки (-) обох LED -> **GND**.

## Процедури в коді
- `PICC_IsNewCardPresent()` / `PICC_ReadCardSerial()`: Виявлення мітки в полі антени.
- `PCD_Authenticate(...)`: Перевірка прав доступу до блоку.
- `MIFARE_Read(...)`: Отримання 16 байт з обраного блоку.
- `MIFARE_Write(...)`: Запис 16 байт даних (наприклад, "LPNU_LAB_2026_OK").

> [!CAUTION]
> Не подавайте 5V на VCC модуля MFRC522, це може вивести його з ладу. Використовуйте тільки піни 3.3V.
